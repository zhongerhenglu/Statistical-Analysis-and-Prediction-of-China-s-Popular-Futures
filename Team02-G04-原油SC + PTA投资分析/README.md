# Team02-G04：原油SC + PTA投资分析

## 1. 研究主题与决策主体

- 研究主题：能源化工（原油SC + PTA）
- 决策主体：
  - 能源化工行业投资者（期货交易员、商品基金经理）
  - 下游聚酯企业采购与风险管理部门
  - 原油和PTA套利/套期保值策略设计者

本分析旨在回答：在当前宏观与行业背景下，原油（SC）与PTA价格的相关关系、波动特征、风险集中期如何影响下游成本与投资策略？

## 2. 背景与研究动机

- PTA 是聚酯产业链中的核心原料，价格逻辑高度依赖原油及其下游化工产品。原油价格波动对 PTA 成本和市场供需传导具有直接影响。
- 能源化工市场在 2020-2025 年期间经历了周期性震荡、宏观政策调整与全球供应链变化，分析价格传导关系对投资与采购决策尤为重要。
- 课程要求强调：明确“决策主体”“政策/背景”“统计事实”。本项目重点聚焦“做决策的主体是谁、他们面临什么样的风险与机会、哪些数据可以支撑结论”。

## 3. 数据来源与处理

### 3.1 数据来源

- 原油（SC）主力合约：`SC0`
- PTA 主力合约：`TA0`
- 时间范围：2010-01-01 至 2025-12-31
- 数据来源：`akshare` 接口获取，原始文件保存在 `data_raw/`

### 3.2 数据清洗

清洗步骤由 `02_data_clean.py` 执行：
- 统一日期格式
- 去重并按日期排序
- 过滤无效价格和成交量数据
- 计算对数收益率 `ret` 和 20 日滚动波动率 `vol20`
- 对原油与 PTA 数据做时间对齐（inner join）

### 3.3 主要输出文件

- `data_clean/final.csv`：清洗后对齐后的数据集
- `analysis_summary.txt`：关键结论摘要
- `plots/`：可视化图表输出

4. 相关性分析（价格、收益率、波动率）

## 5. 核心结论

从 `analysis_summary.txt` 提取的结论如下：

1. 价格关系
   - SC 和 PTA 价格具有较强正相关性（约 0.71）。
   - 原油是 PTA 长期价格趋势的主导驱动因素。

2. 波动特征
   - 两者均展现波动率聚集与厚尾分布特征。
   - SC 的平均波动率（0.0213）高于 PTA（0.0151）。

3. 风险特征
   - SC 发生极端波动（收益率绝对值 > 3%）的频率为 14.47%。
   - PTA 发生极端波动的频率为 7.82%。

4. 交易与风险管理建议
   - 当 SC 显示明确趋势时，PTA 趋势交易更可靠。
   - 高波动期应适当降低持仓并加强风险控制。
   - 在原油价格高位时，建议采用套期保值策略以控制下游成本风险。
