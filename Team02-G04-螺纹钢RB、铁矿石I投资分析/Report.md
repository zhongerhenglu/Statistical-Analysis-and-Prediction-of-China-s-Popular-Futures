# 地产基建期货（螺纹钢RB、铁矿石I）量化分析报告
## Team02‑G04｜数据分析与经济决策 ds2026

# 1 研究背景与决策主体
## 1.1 决策主体
1. **钢厂企业**：铁矿石采购成本管控、原料套期保值
2. **基建/地产企业**：建材采购、成本风险对冲
3. **期货交易者**：趋势判断、仓位管理、波动率风控
4. **贸易商**：跨品种套利、库存周期决策

## 1.2 政策与市场背景
螺纹钢与铁矿石构成我国地产基建核心产业链：铁矿石是炼钢主要原料，螺纹钢直接用于建筑施工。
近年地产政策调整、基建投资波动、铁矿石进口格局变化，导致两品种价格联动明显、波动加剧。
传统分析多依赖经验，缺少量化支撑，本研究通过日度行情数据挖掘波动规律与联动关系，提供数据驱动的决策依据。

## 1.3 研究范围
- 品种：螺纹钢RB主力合约、铁矿石I主力合约
- 时间：2010‑01‑01 ~ 2025‑12‑31
- 频率：日度行情
- 数据来源：akshare公开免费接口

# 2 数据获取与清洗
## 2.1 数据字段
date, open, high, low, close, volume, amount, oi

## 2.2 数据获取
使用`01_data_collect.py`获取原始行情，统一英文列名，避免中文格式错误。

## 2.3 标准化清洗流程（课程规范）
1. 时间格式标准化
2. 去重，剔除重复交易日
3. 过滤价格、成交量异常值
4. 缺失值删除
5. 计算日对数收益率、20日滚动波动率
6. 两品种内连接，对齐共同交易日

## 2.4 数据输出
- 原始数据：`data/data_raw/`
- 清洗后数据：`data/data_clean/final.csv`，无缺失、无异常

# 3 描述性统计
- 螺纹钢与铁矿石价格、收益率高度相关
- 铁矿石波动率整体高于螺纹钢，风险更大
- 收益率均呈现**尖峰厚尾、波动聚集**特征，极端行情概率更高

# 4 可视化分析
## 4.1 价格时序图
两品种长期趋势高度一致，铁矿石先行波动，螺纹钢跟随，体现上游成本传导逻辑。地产下行、基建发力阶段出现阶段性分化。

## 4.2 收益率直方图与核密度
收益率不服从正态分布，极端涨跌事件多于理论预期，尾部风险显著。

## 4.3 箱线图
铁矿石异常值更多，波动更剧烈。

## 4.4 20日波动率时序
高波动集中在政策出台、宏观周期切换阶段，存在明显波动聚集。

## 4.5 相关性散点与热力图
- 价格相关系数>0.75，强正相关
- 收益率、波动率同步性高，成本传导稳定

# 5 核心结论与决策建议
## 5.1 量化结论
1. 螺纹钢与铁矿石**高度正相关**，铁矿石是螺纹钢核心成本驱动。
2. 两品种均存在**波动聚集、尖峰厚尾**，极端风险不可忽视。
3. 铁矿石波动大于螺纹钢，对宏观政策更敏感。
4. 地产基建周期直接决定双品种中长期趋势。

## 5.2 落地建议
1. **交易者**：铁矿石趋势明确时顺势交易螺纹钢；波动率抬升时降低仓位。
2. **钢厂**：铁矿石高位区间提前买入套保，锁定原料成本。
3. **基建企业**：螺纹钢低位批量采购，对冲未来涨价风险。

# 6 局限性
1. 未纳入港口库存、钢厂开工率、地产销售等产业高频数据。
2. 仅做统计描述，未构建预测模型。
3. 主力合约换月存在小幅基差扰动。
